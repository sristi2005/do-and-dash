# Do and Dash – AI Assisted Dashboard

A modern productivity dashboard built using **React, TypeScript, and Tailwind CSS**, scaffolded with the help of **AI development tools (Lovable)** to accelerate frontend development and UI prototyping.

## Overview

Do and Dash is a dashboard-style web application designed to help users manage daily tasks, goals, and productivity workflows in one place. The project demonstrates how AI-assisted developer platforms can speed up the process of building structured frontend applications.

## Features

* Authentication pages for secure user access
* Dashboard layout with modular UI components
* Task management and goal tracking sections
* Social feed style interaction components
* Responsive design using Tailwind CSS
* Scalable component architecture

## Tech Stack

Frontend

* React
* TypeScript
* Tailwind CSS
* Vite

Backend / Services

* Supabase (authentication & database)

Development Tools

* Lovable AI platform (for rapid UI scaffolding)
